# GOKAMS 한국미술 다국어 용어사전 분석

> 분석일: 2025-01-25
> 대상: https://www.gokams.or.kr:442/visual-art/art-terms/

---

## 1. 개요

한국예술경영지원센터(KAMS)에서 운영하는 한국미술 다국어 용어사전.
20세기 한국미술 관련 용어, 인명, 단체/기관명을 5개 언어로 제공.

### 데이터 현황

| 카테고리 | 항목 수 | 제공 언어 |
|---------|--------|----------|
| 미술용어 | ~157개 | 한/영/한문/중/일 |
| 인명 | ~157개 | 한/영/한문 |
| 단체/기관명 | ~157개 | 한/영/한문/중/일 |

---

## 2. 장점

### 2.1 다국어 병렬 제공
- 5개 언어 동시 표기 (한국어, 영어, 한문, 중국어, 일본어)
- 번역가/연구자에게 즉시 참조 가능한 형태

### 2.2 한자 표기 포함
- 인명의 한자 표기 제공
- 동아시아 맥락에서 중요한 정보

### 2.3 해제(정의) 제공
- 미술용어에 상세 설명 포함
- 4개 언어로 해제 번역 제공

### 2.4 번역 표기 권고안
- 별도 섹션으로 번역 원칙 설명
- 음역/의역 선택 기준 제시
- 예: "단색화 → Dansaekhwa" (음역 권고)

### 2.5 공신력 있는 출처
- 국립현대미술관, DA-Arts, 김달진미술연구소 DB 기반
- 국립국어원 감수

### 2.6 초성 검색
- ㄱ~ㅎ 초성별 필터링 가능

## 3. 단점

### 3.1 데이터 접근성

| 문제 | 설명 |
|------|------|
| API 비공개 | 프로그래밍적 접근 불가 |
| JavaScript 링크 | `javascript:go_view('3')` - URL 직접 공유 불가 |
| 데이터 내보내기 없음 | CSV/JSON 다운로드 기능 없음 |
| 인코딩 오류 | 일부 상세 페이지 깨짐 (인명 상세) |

### 3.2 데이터 품질

| 문제 | 예시 |
|------|------|
| 영문 표기 불일치 | `KANG Kyungkoo` vs `Kang Kukjin` (대소문자) |
| 성/이름 구분 없음 | 단일 문자열로만 제공 |
| 중문/일문 번역 누락 | 일부 셀 비어있음 |
| 일부 영문 누락 | `강진희 → - → 姜璡熙` |

### 3.3 범위 제한

| 제한 | 내용 |
|------|------|
| 시대 | 20세기 한정 (1850-1970 출생) |
| 동시대 작가 | 1970년 이후 출생 미포함 |
| 항목 수 | ~157개 (실무 활용에 부족) |

### 3.4 UX 문제

| 문제 | 설명 |
|------|------|
| 전체 목록 탐색 어려움 | "더보기" 클릭 필요, 페이지네이션 없음 |
| 상호참조 없음 | 관련 용어/인물 간 링크 부재 |
| 갱신 이력 없음 | 마지막 업데이트 시점 불명 |
| 검색 결과 URL 없음 | 특정 항목 직접 링크 불가 |

### 3.5 구조적 문제

| 문제 | 설명 |
|------|------|
| 데이터 중복 | 같은 항목이 "미술용어"와 "권고안"에 이중 등재 |
| 카테고리 경계 모호 | 일부 용어의 분류 기준 불명확 |
| 출처 추적 불가 | 개별 항목의 원출처 미표시 |

## 4. Korean Art Lexicon이 참고할 인사이트

### 4.1 채택할 점

| GOKAMS 장점 | 우리 적용 방안 |
|-------------|---------------|
| 다국어 병렬 | 한/영 우선, 확장 가능 구조 |
| 한자 표기 | `name.hanja` 필드 추가 고려 |
| 번역 권고안 | 출처별 표기 + 권고 표기 구분 |
| 공신력 출처 | sources 필드에 기관 명시 |

### 4.2 개선할 점

| GOKAMS 단점 | 우리 해결책 |
|-------------|------------|
| API 비공개 | 오픈소스, GitHub 직접 접근 |
| JavaScript 링크 | 정적 파일, 고유 URL 제공 |
| 데이터 내보내기 없음 | YAML/JSON 직접 다운로드 |
| 영문 표기 불일치 | 스키마 검증 + 리뷰 프로세스 |
| 성/이름 구분 없음 | `name.latn.family`, `name.latn.given` 분리 |
| 출처 추적 불가 | `sources.name_used`로 출처별 표기 추적 |
| 갱신 이력 없음 | Git 히스토리로 모든 변경 추적 |
| 1970년 이후 미포함 | 시대 제한 없음 |

## 5. 스키마 개선 제안

### 5.1 한자 필드 추가

```yaml
name:
  ko:
    full: 김환기
    family: 김
    given: 환기
    hanja: 金煥基  # 추가
  latn:
    preferred: Kim Whanki
    family: Kim
    given: Whanki
```

### 5.2 번역 유형 구분

GOKAMS의 번역 분류 체계:
- **음역**: 한국어 발음 그대로 (Dansaekhwa)
- **대응역**: 기존 영어 용어 대응 (figurative)
- **직역**: 의미 그대로 번역 (moon jar)
- **의미역**: 설명적 번역 (painting on traditional Korean architecture)

```yaml
# 미술용어 스키마 확장 시 고려
term:
  ko: 단색화
  en:
    preferred: Dansaekhwa
    type: transliteration  # 음역
    alternatives:
      - form: "monochrome painting"
        type: semantic  # 의미역
```

### 5.3 관련 항목 연결

```yaml
# 상호참조 추가
related:
  - person.lee-ufan
  - term.dansaekhwa
  - exhibition.korean-monochrome-painting-1970s
```

## 7. 결론

### GOKAMS의 가치
- 공신력 있는 기관의 번역 표준 시도
- 다국어 병렬 데이터의 선례

### GOKAMS의 한계
- 폐쇄적 접근성
- 데이터 품질 불일치
- 범위 제한 (20세기, ~157개)

### Korean Art Lexicon의 차별점

| 측면 | GOKAMS | Korean Art Lexicon |
|------|--------|-------------------|
| 접근성 | 폐쇄 | 오픈소스 |
| 데이터 형식 | HTML only | YAML/JSON |
| 출처 추적 | 불가 | 출처별 표기 기록 |
| 버전 관리 | 없음 | Git 기반 |
| 범위 | 20세기 | 제한 없음 |
| 기여 | 불가 | PR 기반 커뮤니티 |
| 성/이름 | 단일 문자열 | 분리 구조 |
